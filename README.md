# 🚀 Rinex Website

A modern and responsive internship & course platform designed to help students explore opportunities and kickstart their tech careers.

---

## 🌟 Features
- 🎯 Clean and modern UI design  
- 📱 Fully responsive (mobile + desktop)  
- 📚 Course listing with enrollment option  
- 📝 Interactive enrollment form  
- ⚡ Smooth scrolling and animations  
- 📍 Contact section with location integration  

---

## 🛠️ Tech Stack
- HTML5  
- CSS3  
- Bootstrap 5  
- JavaScript
